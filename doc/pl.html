(static) nodeTreeGenerator(nodeTreeArray, __parentopt) → {Array.<Element>}

Will generate some DOM and eventually apend to an Element
Parameters:
Name Type Attributes Description
nodeTreeArray Array.<NodeTree> Array of NodeTree
__parent Element <optional>
 if precise, each NodeTree of nodeTreeArray will be appended to him
Author:
  • Purexo
Source:
  • dom.js, line 158
See:
  • NodeTree
Throws:
pl.exceptions.NotArrayException | pl.exceptions.NotStringException | pl.exceptions.NotElementException
Returns:
Type
Array.<Element>
Example
pl.nodeTreeGenerator([{}])
 // return [<div></div>]

pl.nodeTreeGenerator([{
 tag: 'p',
 text: 'Hello World',
 attr: [
   {name: 'className', value: 'foo bar'},
   {name: 'data-foo', value: 'bar'}
 ],
 in: [
   {},
   {
     tag: span,
     text: 'Inner Hello World'
   }
 ]
}], pl.$('#test'))

// return
// <p class="foo bar" data-foo="bar">
//   Hello World
//   <div></div>
//   <span>Inner Hello World</span>
// </p>

// will update Element with id test
// <div id="test">
//   <\!-- already present content -->
//   <p class="foo bar" data-foo="bar">
//     Hello World
//     <div></div>
//     <span>Inner Hello World</span>
//   </p>
// </div>
