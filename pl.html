__parent Collection <optional>

Author:

See:
  • NodeTree
Throws:
pl.Exception.NotArrayException | pl.Exception.NotStringException | pl.Exception.NotElementException
Returns:
Type
Collection
Example
pl.generate([{}])
 // return [<div></div>]

pl.generate([{
 tag: 'p',
 text: 'Hello World',
 attr: [
   {name: 'className', value: 'foo bar'},
   {name: 'data-foo', value: 'bar'}
 ],
 in: [
   {},
   {
     tag: span,
     text: 'Inner Hello World'
   }
 ]
}], pl('#test'))

// return
// <p class="foo bar" data-foo="bar">
//   Hello World
//   <div></div>
//   <span>Inner Hello World</span>
// </p>

// will update Element with id test
// <div id="test">
//   <\!-- already present content -->
//   <p class="foo bar" data-foo="bar">
//     Hello World
//     <div></div>
//     <span>Inner Hello World</span>
//   </p>
// </div>
